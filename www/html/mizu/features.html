Features

Plug-and-play

Plain HTML and JavaScript. No custom language, build system or configuration required.

Just include the library to get started.

Cross-platform

Compatible with most JavaScript and TypeScript runtimes, including browsers.

Any-side

Render content server-side, client-side, or your own way through [user-friendly APIs](#api-user).

statically, dynamically,

Hackable

Cherry-pick features and create your own setup easily through [developer-friendly APIs](#api-dev) or using our [custom builder](/build).
