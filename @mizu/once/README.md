# `*once`

| Version                               | Phase                 |
| ------------------------------------- | --------------------- |
| ![](https://jsr.io/badges/@mizu/once) | 99 — `POSTPROCESSING` |

Render an element once and skip subsequent updates.

```html
<div *once>
  <!--...-->
</div>
```

## Modifiers

### `.flat[boolean]`

Replace this element by its content.
