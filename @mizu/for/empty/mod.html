*empty

Conditionally render an element after a *for directive.

Must be placed immediately after an element with a *for or another *empty directive. Elements generated by the *for directive do not apply to this restriction. There is currently no special handling for <template> elements, but future versions may introduce specific behavior for these elements. number The number of elements generated by the preceding *for directive. This value may differ from the actual number of iterations processed if conditional directives were applied. Inverts the condition, rendering the element when at least one element is generated.
