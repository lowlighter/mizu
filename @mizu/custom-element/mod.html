*custom-element="tagname"

Register a new custom element.

Must be defined on a <template> element. The tagname must be a valid custom element name. Valid custom element names may be specified as is. Custom elements registered this way do not use Shadow DOM, their content is rendered directly. Replace occurrences of this custom element with its content. Note that $slots and $attrs variables are not accessible when using this modifier. Record<PropertyKey, HTMLSlotElement> A record of #slot elements by <slot> name. The unnamed slot is accessible using $slots[""]. Record<PropertyKey, string> A record of HTML attributes specified on the custom element. #slot

Specify target <slot> in an element defined by a *custom-element directive.

    • Elements without a #slot directive are appended to the default (unnamed) slot. Elements targeting the same slot are appended in the order they are defined.
