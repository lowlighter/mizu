*custom-element="tagname"

Register a new custom element.

Must be defined on a <template> element. Specified tagname must satisfy the definition of a valid name. Valid custom element names may be specified as is. Custom elements registered this way do not use Shadow DOMs but rather have their content rendered directly. Replace occurences of this custom element by their content. Note that it not possible to access $slots and $attrs variables when using this modifier. Record<PropertyKey, HTMLSlotElement> A record of specified #slot elements by <slot> name (unamed slot is accessible using $slots[""]). Record<PropertyKey, string> A record of specified attributes on the custom element. #slot

Specify target <slot> in an element defined by a *custom-element directive.

    • Elements without a #slot directive are appended to the (unamed) default slot. Elements targetting a same slot are all appended to it in the same order they were defined.
